<h2 id="1">اتم ها</h2>
<br>
<p>

‎در سال گذشته آموختیم که تمام مواد از اتم ها تشکیل شده اند و امسال قصد داریم در مورد خود اتم ها و ذرات سازنده آنها صحبت کنیم

<br> 

‎اتم ها از سه ذره اصلی تشکیل شده اند الکترون ها، پروتون ها و نوترون ها تشکیل شده اند الکترون ها بار منفی، پروتون ها بار مثبت و نوترون ها بدون بار هستند پروتون ها و نوترون ها در هسته و الکترون ها در لایه هایی حاوی انرژی در حال گردش به دور اتم هستند

<br> 
</p>
<h2 id="2">جدول تناوبی</h2>
<br>
<p>
‎برای نشان دادن عناصر از جدول تناوبی استفاده می کنیم که 118 عنصر در آن موجود می باشد علاوه بر تقسیم بندی عناصر در جدول تناوبی اطلاعاتی در مورد اتم نیز در آن موجود می باشد برای نشان دادن عناصر در جدول تناوبی از نشان شیمیایی آنها استفاده می کنیم (برای مثال برای هیدروژن از H و برای اکسیژن از O استفاده می کنیم)

<br> 
</p>
<h2 id="3">عدد اتمی</h2>
<p>
‎یکی از اطلاعات عدد اتمی می باشد که همان تعداد پروتون ها می باشد باید به این نکته توجه داشت که تغییر تعداد پروتون های یک عنصر غیر ممکن می باشد به همین دلیل تغییر عناصر به یکدیگر نیز امکان پذیر نمی باشد 

‎یکی دیگر از اطلاعات عدد جرمی می باشد با توجه به این که اکثر جرم اتم مربوط به هسته می باشد برای محاسبه عدد جرمی نیز کافی است محتویات هسته را محاسبه کنیم به عبارتی دیگر مجموع تعداد پروتون ها و نوترون ها عدد جرمی می باشد.

<br> 

‎یکی دیگر از اطلاعات یون یا خنثی بودن اتم می باشد در واقع اتم زمانی تبدیل به یون می شود که از نظر بار الکتریکی خنثی نباشد به عبارتی دیگر کمبود یا زیاد بودن الکترون باعث می شود که اتم یون شد. 



</p>

$(document).ready(function(){ 

// Add smooth scrolling to all links 

$("link").on('click', function(event) { 



// Make sure this.hash has a value before overriding default behavior 

if (this.hash !== "") { 

// Prevent default anchor click behavior 

event.preventDefault(); 



// Store hash 

var hash = this.hash; 



// Using jQuery's animate() method to add smooth page scroll 

// The optional number (800) specifies the number of milliseconds it takes to scroll to the specified area 

$('html, body').animate({ 

scrollTop: $(hash).offset().top 

}, 800, function(){ 



// Add hash (#) to URL when done scrolling (default click behavior) 

window.location.hash = hash; 

}); 

} // End if 

}); 

}); 

</script>
